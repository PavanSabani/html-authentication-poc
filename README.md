# Authentication System

## Project Overview

This project is a **styled authentication system** built using **HTML, Bootstrap 5, and custom CSS**.

It simulates a complete authentication flow with:

- Login
- Registration
- Forgot Password
- Reset Password
- Dashboard

All pages are connected using navigation and enhanced with modern UI design.

---

## Features

### Core Features

- Login system UI
- Registration form
- Forgot & Reset password flow
- Dashboard with navbar

### UI Enhancements

- Bootstrap 5 integration
- Responsive design (mobile + desktop)
- Modern card-based layout
- Gradient background (Blue → Purple)
- Google Fonts (Poppins)

### Advanced Features (Bonus)

- Show/Hide password toggle 👁️
- Password strength indicator 🔒
- Smooth hover animations
- Input focus glow effects
- Responsive navbar

---

## Technologies Used

- HTML5
- Bootstrap 5
- CSS3
- JavaScript (for navigation + interactions)
